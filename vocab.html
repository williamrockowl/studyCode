Phase 1:

Variable: A container in which we can store values for later retrieval.

String Concatenation: A way to take two strings and add one to the other creating a single string. The way to concatenate strings in JavaScript is with the + operator.

String Interpolation: A way to insert values in the middle of a string.

.slice() Method: Extracts a section of a string and returns it as a new string, without modifying the original string.
